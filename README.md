# Snake Game

(linux only)

![til](./gif.gif)

## Building
After cloning the repository, run:
```bash
cd snake-terminal
make
```
This will build snake terminal and create the binary at `./main`

## How to Play
### Commands
`q` (quit), `p` (pause)

### Movement
`w` (up), `s` (down), `a` (left), `d` (right)
